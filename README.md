# Calculator Program in C

## Project Overview

This project is a simple menu-driven calculator developed in C as part of the CodeAlpha C Programming Internship.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero validation
- Menu-driven interface
- Repeated calculations until the user exits

## Technologies Used

- C Programming
- Functions
- Switch Case
- Loops
- Conditional Statements

## How to Run

1. Compile:
   gcc calculator.c -o calculator

2. Run:
   ./calculator

## Internship

This project was completed as **Task 1** of the **CodeAlpha C Programming Internship**.
